⚙️ Parâmetros dos Widgets

Widget Benefícios de Saúde (health_benefits_widget.php)

Parâmetro Valores Padrão Descrição
limit 1-10 4 Número de posts a exibir
theme benefits, compact benefits Tema visual do widget
description true, false true Mostrar descrição dos posts

Widget Posts Tradicionais (latest_posts_widget.php)

Parâmetro Valores Padrão Descrição
limit 1-10 5 Número de posts a exibir
theme default, minimal, card default Tema visual do widget
excerpt true, false true Mostrar resumo dos posts
date true, false true Mostrar data de publicação
category true, false true Mostrar categoria do post
